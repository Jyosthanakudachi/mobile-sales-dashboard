# Mobile Sales Dashboard

## Project Overview

The Mobile Sales Dashboard is an interactive Power BI project designed to analyze mobile sales performance across different cities, brands, and products. The dashboard provides insights into sales trends, revenue generation, customer purchasing behavior, and key performance indicators (KPIs) to support data-driven decision-making.

## Features

* Interactive Power BI dashboard
* Sales performance analysis
* Brand-wise and city-wise sales insights
* Revenue and quantity sold tracking
* KPI monitoring and trend analysis
* Dynamic filters and visualizations
* Data modeling and DAX calculations

## Tools & Technologies

* Power BI
* DAX
* Data Modeling
* Data Visualization
* Microsoft Excel

## Dataset

The dataset contains over 1000+ sales records, including:

* Mobile Brand
* City
* Sales Quantity
* Revenue
* Customer Information
* Product Details

## Key Insights

* Identified top-performing mobile brands and cities.
* Analyzed sales trends and revenue patterns.
* Tracked KPIs to evaluate overall business performance.
* Generated actionable insights through interactive visualizations.

## Business Objective

To provide a comprehensive view of mobile sales performance, helping businesses identify growth opportunities, optimize sales strategies, and make informed decisions.

## Project Outcome

Developed an interactive dashboard that simplifies sales analysis, improves reporting efficiency, and enables stakeholders to monitor performance through real-time visual insights.
